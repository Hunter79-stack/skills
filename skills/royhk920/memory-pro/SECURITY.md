# Security & Privacy

## Summary
Memory‑Pro operates **locally only**. It is designed for personal knowledge management and recall within OpenClaw.

## What this skill does NOT do
- ❌ No outbound network requests
- ❌ No data exfiltration
- ❌ No background daemon or persistence outside user control
- ❌ No automatic execution without user intent

## Data Handling
- All memory data is stored locally on the user's machine
- No uploads to third‑party services
- Users can delete memory data at any time

## Execution Model
- Scripts are executed only when explicitly triggered by the user
- No hidden or scheduled execution

This document is provided to improve transparency for users and automated security review systems.
