# Common Composio Tool Reference

Quick-reference for frequently used tools. Always confirm exact tool names with `composio.COMPOSIO_SEARCH_TOOLS` first.

## Email (Gmail, Outlook)

| Task | Tool |
|------|------|
| Send email | `composio.GMAIL_SEND_EMAIL` |
| Read inbox | `composio.GMAIL_FETCH_EMAILS` |
| Search emails | `composio.GMAIL_SEARCH_EMAILS` |
| Reply to email | `composio.GMAIL_REPLY_TO_EMAIL` |
| Send via Outlook | `composio.OUTLOOK_SEND_EMAIL` |
