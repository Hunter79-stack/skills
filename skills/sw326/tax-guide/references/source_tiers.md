# 출처 신뢰도 티어

> 공통 Tier 구조 + 인용 형식: [`../../common/source_tiers_base.md`](../../common/source_tiers_base.md)

## Tier 1 — 최고 신뢰도 (공식)
- **국세청** (nts.go.kr) — 세율표, 고시, 공식 가이드
- **법제처** (law.go.kr) — 소득세법, 부가가치세법, 상속세 및 증여세법, 지방세법
- **기획재정부** — 세법 개정안, 예산안
- **국세청 홈택스** (hometax.go.kr) — 신고 절차, 서식
- **찾기쉬운 생활법령** (easylaw.go.kr) — 일반인 대상 세금 설명

## Tier 2 — 높은 신뢰도 (전문 매체)
- **한국세무사회** / **한국공인회계사회**
- **조세일보** / **세정일보** — 세무 전문 매체
- **한국경제** / **매일경제** 세금 섹션

## Tier 3 — 보강용
- 세무사 사무소 블로그 (법인 운영 확인된 경우)
- 세금 계산기 서비스 (결과 참고만, 단독 인용 불가)

## 금지 출처
- 개인 블로그, 네이버 지식iN → 인용 금지
- 유튜브 세금 영상 → 개념 확인만, 인용 불가
- 포털 뉴스 요약 → 원문 Tier 1/2 링크로 대체

## 인용 형식
```markdown
# 법령
[소득세법](https://www.law.go.kr/법령/소득세법) 제XX조 (시행 YYYY.MM.DD)

# 국세청 공식 자료
[국세청](https://www.nts.go.kr) — 자료명

# API
(law-search: target=법령, query=소득세법 제XX조)

# 추정

```
