# Code Flow Visualizer

将代码转换为 Mermaid 流程图表示，便于理解代码逻辑流程。

## 功能

- 分析代码逻辑结构
- 生成 Mermaid 流程图
- 支持 Python、JavaScript、TypeScript

## 使用方式

当用户需要可视化代码流程时说"可视化代码"或"生成流程图"。

## 示例

```
用户: 帮我把这个Python函数生成流程图
助手: (使用 code_flow_visualizer skill)
```

## 工具

使用 Mermaid JS 或 PlantUML 生成图表。

## 限制

- 仅处理单一函数/方法
- 不支持 goto 语句
- 复杂循环可能简化显示
