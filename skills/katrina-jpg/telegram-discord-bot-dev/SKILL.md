# Telegram/Discord Bot Developer

幫你整Telegram同Discord Bot既Skill

## 功能
1. **Bot開發** - 整各種function既bot (trading, gaming, utility)
2. **Webhook整合** - 對接API、數據、notifications
3. **自動化** - 設定自動回覆、scheduled messages
4. **Analytics** - 追蹤用戶行為同 engagement

## 安裝
```bash
npx clawhub@latest install telegram-discord-bot-dev
```

## 使用
當用戶話「整Bot」既時候：
1. 問佢想要咩功能
2. 建議技術方案
3. 提供報價同development timeline

## 範例
```
User: 我想要一個crypto price alert bot
AI: 你想邊種通知方式？Telegram定Discord？
```
