# Data Analysis Suite

## What This Skill Does
Analyzes CSV/Excel data and generates insights with charts.

## Features
- CSV/Excel reader
- Statistical analysis
- Chart generator (bar, line, pie)
- Trend detection
- Report builder (PDF/HTML)

## Usage
```python
from data_analyzer import Analyzer

analyzer = Analyzer("sales_data.csv")
insights = analyzer.insights()
charts = analyzer.charts()
report = analyzer.report(format="pdf")
```
