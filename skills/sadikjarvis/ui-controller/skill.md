# UI Controller

## Overview
UI Controller is an OpenClaw skill that allows controlling local user interface
actions on the machine where the OpenClaw gateway is running.

It is intended for personal automation and desktop control.

## What this skill does
- Triggers UI related actions
- Controls local desktop behaviour
- Runs only on the gateway machine

## Usage
Send a message that matches the configured pattern in the task file.

## Platform
Windows

## Security notice
This skill can control the local desktop.
Use only in a trusted environment.
