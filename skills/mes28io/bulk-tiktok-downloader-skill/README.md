# bulk-tiktok-downloader-skill

OpenClaw skill for bulk TikTok video downloads using yt-dlp.

## Includes
- `SKILL.md`
- `scripts/downloader.py`
- `scripts/requirements.txt`
- `references/upstream-readme.md`

## Quick run
```bash
python3 -m pip install --user -r scripts/requirements.txt
python3 scripts/downloader.py urls.txt downloads
```
