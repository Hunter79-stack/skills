# 你提的 3 点优化，在 v3.1 的落地方式

1) 关键词颜色突出：
- 强制“关键词高亮策略”出现在 Final Prompt 中（不同颜色/色块/更粗描边/发光）
- 要求与非关键词形成清晰层级

2) 关键词轻微弧度：
- 强制“轻弧排版策略”出现在 Final Prompt 中
- 规则限定为“轻微弧度”，并给出默认方向（左下→右上微上扬）

3) 背景不写死蓝色：
- 引入“主题配色方案”：主色/辅色/高亮色三元组
- 背景纹理仍保持系列质感，但主色随主题变化
