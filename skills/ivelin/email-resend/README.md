Send and receive emails using the Resend API.

## Features

- Send emails directly via Resend API
- Receive email notifications via cron
- Draft replies with proper threading
- Download attachments from inbound emails

## Setup

```bash
export RESEND_API_KEY="re_123456789"
```

## Usage

Invoke via OpenClaw when you need to send or receive emails.
