SJ-IRAC Non-Use (撤三) Cross-Examination Library
Version: 1.0.0
Last Updated: 2026-02-03

Purpose
Provide examiner-oriented, evidence-based cross-examination statements for challenging registrant’s use evidence in non-use cancellation proceedings.
Hard Constraint: No fabrication. No speculation. Every point must be tied to a specific exhibit and a specific defect type.

Usage Rule
For each exhibit, select:
- Defect Type (A-F)
- Ask 1–3 Key Questions
- Conclude with "cannot prove" wording (objective, non-emotional)
Always reference: Exhibit ID + date (if shown) + carrier (invoice/contract/screenshot/etc.)

========================
A. Authenticity / Source Defects (真实性/来源瑕疵)
========================

A1. Original / verification missing
- 该证据未见原件或可核验的来源信息，无法排除事后补制或内容被修改的可能，真实性不足以作为使用证据采信。
- 该材料缺乏可供核验的出具主体、形成过程或对应原始记录，无法证明其为交易过程中自然形成的真实凭证。

Key Questions
- 是否提供原件/原始电子文件及生成路径（后台记录/流水/系统导出）？
- 是否可提供出具主体的真实性核验材料（税控/系统校验/第三方平台记录）？

A2. Stamp/signature inconsistency
- 证据上所载签章/签名信息与其他材料不一致，且无合理解释，真实性存疑。
- 同一交易链条中主体签章、格式与编号规则明显不统一，缺乏连续性与自然形成痕迹。

Key Questions
- 签章主体与合同相对方是否一致？是否有授权链？
- 编号规则、格式是否与该主体日常票据/合同习惯一致？

A3. Self-produced evidence without corroboration
- 该证据属于权利人单方制作材料，缺乏第三方或客观记录印证，单独不足以证明商标真实商业使用。

Key Questions
- 是否有第三方记录可交叉验证（物流、税控、平台订单、付款流水）？

========================
B. Relevance Defects (关联性瑕疵)
========================

B1. Goods/services unclear
- 证据所载商品/服务名称、规格或描述不清，无法对应核定使用商品/服务，关联性不足。
- 证据仅能证明发生某种交易或宣传行为，但无法证明系针对核定商品/服务的商标使用。

Key Questions
- 商品/服务描述能否与核定项目逐一对应？是否有型号、SKU、清单、服务内容说明？
- 是否存在“同名不同物”或“非核定范围”的情形？

B2. Mark not shown or not linked to the goods
- 证据未能体现争议商标标识，或无法证明该标识系用于涉案商品/服务之上，难以构成商标法意义上的使用证明。
- 即便材料中出现文字/图形，也缺乏与具体商品、交易或服务提供行为的绑定关系，不能证明商标使用。

Key Questions
- 商标是否在商品包装、标签、服务页面、交易凭证中明确呈现？
- 是否能证明该标识与交易标的为同一对象？

B3. Subject mismatch (user identity)
- 证据所载使用主体与商标注册人不一致，且未见许可、授权或实际控制关系证明，无法证明系权利人或其合法授权主体的使用。

Key Questions
- 是否提供许可合同/授权声明及备案或相应证明？
- 使用主体与权利人的关系链如何形成、是否在使用期内有效？

========================
C. Temporal Defects (时间效力瑕疵)
========================

C1. Outside the use period
- 证据形成时间不在指定使用期间内，无法证明解释期内的连续使用事实。
- 该材料即便能证明其他时期存在行为，也不能替代指定期间内的使用证明。

Key Questions
- 材料形成日期是否明确可核验？是否可证明在指定期间内发生？
- 是否存在时间戳缺失、可编辑、无法溯源的问题？

C2. Unclear or editable timestamps
- 截图/网页/电子材料的时间信息缺乏可核验性，且存在可编辑风险，时间效力不足。

Key Questions
- 是否提供后台导出记录、第三方存证、服务器日志、哈希/时间戳存证？
- 是否能证明截图所示内容在指定期间内实际在线存在？

========================
D. Completeness Defects (完整性瑕疵)
========================

D1. Single-link evidence (no chain)
- 证据仅能证明链条中的单一环节（如宣传或合同），缺乏订单/付款/交付等闭环材料印证，难以证明真实商业使用。
- 缺少能将“商标—商品/服务—交易行为—时间”四要素闭合的证据组合，证明力不足。

Key Questions
- 是否存在对应订单、付款、物流、签收、对账等闭环材料？
- 合同是否实际履行？是否有履行痕迹？

D2. Missing attachments / annexes
- 合同/项目材料未提交附件、清单或履行文件，无法确定标的、数量、价格、履行时间，完整性不足。

Key Questions
- 是否提供合同附件、订单明细、交付清单、服务验收材料？

========================
E. Logical / Commercial Reasonableness Defects (逻辑/交易合理性瑕疵)
========================

E1. Unreasonable transaction pattern
- 证据所示交易金额、频次、相对方结构或业务逻辑缺乏合理性，且缺少独立第三方印证，难以采信为真实市场交易。
- 若无进一步证据解释交易背景与履行过程，该材料的商业合理性不足以支撑真实使用认定。

Key Questions
- 相对方为何选择该商品/服务？是否有往来记录、对账、付款凭证？
- 交易是否与权利人经营规模、渠道模式相匹配？

E2. Circular or internal transactions
- 证据显示交易更接近内部流转或关联交易，但缺少面向市场的真实交易特征，难以证明商标在市场中真实使用。

Key Questions
- 是否存在独立交易对手与真实支付？是否有第三方物流/平台记录？

========================
F. Cross-Examination Resistance Defects (抗反问能力不足)
========================

F1. One-question collapse test
- 该证据无法回答“谁在何时就何种商品/服务以何种方式使用了该商标”的核心问题，抗反问能力不足，难以单独或作为主要证据采信。

Standard Closing (choose one)
- 综上，该证据在真实性/关联性/时间效力/完整性方面存在明显缺陷，难以证明指定期间内对核定商品/服务的真实商标使用。
- 综上，该组证据未能形成可核验的商业使用闭环，不能满足撤三案件对商标使用证据的证明要求。
- 综上，相关材料不足以证明商标使用行为在指定期间内真实发生，依法不应予以采信。
