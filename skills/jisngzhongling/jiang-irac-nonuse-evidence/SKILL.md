## Core Framework

### IRAC Structure (Mandatory)

- Issue: define dispute focus
- Rule: statutory + guideline standards
- Application: evidence-to-element mapping
- Conclusion: go / stop-loss / supplement / attack

### SJ-6 Evidence Chain (Mandatory)

Each evidence item is evaluated under:

1. Authenticity
2. Relevance
3. Completeness
4. Temporal validity
5. Logical consistency
6. Cross-exam resistance

All evidence must be time-aligned and purpose-tagged.
